# Heal & Nourish Website

## What this is

Static website for Heal & Nourish — organic, wholefood, vegan food brand. South African. Hosted on GitHub Pages. No build step, no framework, no npm. Push to main and it deploys.

## Tech

- HTML5 + CSS3 (custom properties) + vanilla JS if needed
- Pico CSS via CDN as base (optional)
- Formspree for contact/order forms
- Google Fonts via CDN
- WebP images, max 200KB each
